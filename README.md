## Open Data Battle
Данный репозиторий содержит решение заданий отборочного турнира соревнования [Open Data Battle](https://open-data-battle.geecko.com/).
#### Содержание
1. Задача Airport: предсказание пассажиропотока аэропорта.
2. Задача Covid-19: определение корреляции между частями света по распространению коронавируса.
3. Задача Genger: определение пола человека на основе признаков на фотографиях.
4. Задача News: определение источника по содержимому новости.
5. Задача Paris: нахождение оптимального числа кластеров выборки объектов недвижимости.
6. Задача Predict: настройка нейронной сети для предсказания зависимости заработной платы от опыта сотрудников.
